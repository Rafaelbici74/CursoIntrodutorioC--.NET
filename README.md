# 🚀 API em C# com .NET | Curso Introdutório

Este projeto foi desenvolvido durante meus estudos em C# e .NET, com foco na construção de uma API REST utilizando boas práticas de desenvolvimento backend.

---

## 📌 Sobre o projeto

A aplicação consiste em uma API desenvolvida com **C# e .NET**, permitindo a realização de operações básicas de um CRUD (Create, Read, Update e Delete).

O projeto também utiliza:
- Entity Framework para acesso a dados
- Validação de dados
- Tratamento de exceções
- Documentação com Swagger

O objetivo foi consolidar conhecimentos fundamentais de desenvolvimento backend e estruturação de APIs.

---

## 🧠 O que aprendi

Durante o desenvolvimento deste projeto, pratiquei:

- ✅ Fundamentos da linguagem C#
- ✅ Estrutura de projetos em .NET  
- ✅ Criação de APIs REST com ASP.NET Core  
- ✅ Conceitos de Programação Orientada a Objetos (POO)  
- ✅ Integração com banco de dados utilizando Entity Framework  
- ✅ Validação de dados com FluentValidation  
- ✅ Tratamento de erros e exceções  
- ✅ Uso do Swagger para documentação e testes da API

## ⚙️ Funcionalidades

A API permite:

- 📥 Criar registros  
- 📄 Listar registros  
- ✏️ Atualizar dados  
- ❌ Deletar registros  

Todos os endpoints podem ser testados diretamente via Swagger.
